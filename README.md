# Назва проекту: Алгоритми сортування на Java

Цей проект містить реалізації різних алгоритмів сортування на Java. Алгоритми реалізовані простим та зрозумілим способом для навчальних цілей.

## Початок роботи

Щоб почати роботу з цим проектом, клонуйте репозиторій на свій локальний комп'ютер.

## Передумови

Вам потрібно мати встановлену Java на своєму комп'ютері, щоб запустити цей проект. Ви можете перевірити, чи встановлена Java, виконавши наступну команду в терміналі:

```bash
java -version
```

Якщо Java не встановлена, ви можете завантажити її з офіційного веб-сайту Oracle.

## Запуск проекту

Ви можете запустити кожен алгоритм сортування окремо. Перейдіть до каталогу алгоритму сортування, який ви хочете запустити, та виконайте наступну команду:

```bash
javac FileName.java
java FileName
```

Замініть `FileName` на назву файлу, який ви хочете запустити.

## Запуск проекту в IntelliJ IDEA

1. Відкрийте проект в IntelliJ IDEA.
2. Перейдіть до файлу `src/main/java/com/go/it/QuickSort.java` у вікні інструментів проекту.
3. Клацніть правою кнопкою миші на файлі `QuickSort.java`.
4. Виберіть `Run 'QuickSort.main()'` у контекстному меню.

Це запустить метод `main` у класі `QuickSort`.

## Алгоритми сортування

Цей проект містить наступні алгоритми сортування:

1. **Сортування бульбашкою**: Це простий алгоритм сортування, який постійно проходиться по списку, порівнює сусідні елементи та обмінює їх, якщо вони в неправильному порядку. Прохід по списку повторюється, поки список не буде відсортований.

2. **Швидке сортування**: Швидке сортування - це алгоритм "розділяй та володарюй". Він вибирає елемент як опорний та розділяє заданий масив навколо вибраного опорного елемента. Існує багато різних версій швидкого сортування, які вибирають опорний елемент різними способами.
